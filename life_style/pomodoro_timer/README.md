# Pomodoro Timer

This project provides implementations for a Pomodoro timer using Python. It includes two versions:

- **GUI Version (`pomodoro_timer_gui.py`)**: A graphical user interface (GUI) based Pomodoro timer.
- **Terminal Version (`pomodoro_timer_terminal.py`)**: A terminal-based Pomodoro timer.

## GUI Version

### Files:

- `pomodoro_timer_gui.py`: Python script for the GUI-based Pomodoro timer.
- `pomodoro_timer_gui.bat`: Batch script to install required packages and run the GUI script (for Windows).
- `pomodoro_timer_gui.sh`: Shell script to install required packages and run the GUI script (for Linux).

### Instructions to Run:

- Run `pomodoro_timer_gui.bat` (Windows) or `install_and_run_gui.sh` (Linux) to install the required packages and run the GUI script.

## Terminal Version

### Files:

- `pomodoro_timer_terminal.py`: Python script for the terminal-based Pomodoro timer.
- `pomodoro_timer_terminal.bat`: Batch script to install required packages and run the terminal script (for Windows).
- `pomodoro_timer_terminal.sh`: Shell script to install required packages and run the terminal script (for Linux).

### Instructions to Run:

- Run `pomodoro_timer_terminal.bat` (Windows) or `pomodoro_timer_terminal.sh` (Linux) to install the required packages and run the terminal script.
